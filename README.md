# WATools - WhatsApp Automation Testing Tool

WhatsApp Automation Testing Tool adalah script Python untuk testing login otomatis ke WhatsApp menggunakan emulator Android (BlueStacks, Nox, atau Android Studio) dengan request kode verifikasi SMS.

## ⚠️ DISCLAIMER

**PENTING:** Tools ini hanya untuk tujuan **testing dan development** pada device pribadi Anda. Penggunaan untuk automation massal atau melanggar ToS WhatsApp dapat menyebabkan:
- Ban akun WhatsApp
- Masalah hukum
- Risiko keamanan

Gunakan dengan bijak dan sesuai dengan kebijakan WhatsApp.

## 📋 Requirements

- **Windows 11 Home**
- **Python 3.8 atau lebih tinggi**
- **BlueStacks** (atau Nox Player / Android Studio Emulator)
- **ADB (Android Debug Bridge)** - biasanya sudah termasuk dengan BlueStacks
- **Internet connection** - untuk menerima SMS
